# Nandakumar M Portfolio

Static personal portfolio for GitHub Pages, served on:

```text
nandakumarm.dpdns.org
```

## What Is Included

- Futuristic full-screen hero section with a generated visual asset.
- Responsive navigation for desktop and mobile.
- About, projects, featured work, writings, resources, updates, and contact sections.
- Project and writing filters powered by vanilla JavaScript.
- Static-friendly contact form that opens the visitor's email app.
- No build tools required.

## Edit The Website

Most content lives at the top of `script.js`:

- `SITE` - name, email, location, links, skills.
- `WORKS` - project cards.
- `FEATURED` - highlighted work.
- `WRITINGS` - notes, articles, and thoughts.
- `RESOURCES` - downloads, tools, references, study material.
- `UPDATES` - blog and progress updates.

For detailed instructions, read:

```text
MODIFICATION_GUIDE.md
```

## Deploy

Commit and push changes to the `main` branch. GitHub Pages will update the live site automatically.
