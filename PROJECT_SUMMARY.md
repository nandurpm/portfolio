## 🚀 Quick Start Guide

### Local Testing
```bash
cd personal-portfolio-website
python3 -m http.server 8000
# Visit http://localhost:8000
```

### Customization Steps
1. Update profile image: `assets/images/profile.jpg`
2. Edit personal info in HTML files
3. Update `data/featured.json` with your content
4. Add projects to `data/works.json`
5. Add blog posts to `data/writings.json`
6. Add resources to `data/resources.json`
7. Add updates to `data/updates.json`

### GitHub Pages Deployment
1. Create GitHub repository
2. Push all files to GitHub
3. Enable GitHub Pages in Settings
4. (Optional) Configure custom domain
