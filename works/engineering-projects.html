Smart Portfolio Dashboard

Web Development HTML5 CSS3 JavaScript

Overview

A responsive GitHub Pages dashboard for presenting engineering projects, resume highlights, and blog content with JSON-powered cards. This project demonstrates the power of static site generation combined with intelligent data structures for maintainability.

Key Features

  • JSON-based project and blog data for easy updates
  • Responsive design with CSS Grid and Flexbox
  • Dark/Light theme toggle with persistent storage
  • Smooth animations using AOS (Animate On Scroll)
  • Fast loading - no server required
  • SEO optimized with proper meta tags

Technical Approach

The portfolio uses vanilla JavaScript to load JSON data and dynamically generate card layouts. This approach provides the flexibility of a database-driven site without the maintenance overhead or server costs. All styling is custom CSS with careful attention to accessibility and responsive design.

Deployment

Hosted on GitHub Pages for free, reliable, and version-controlled hosting. The site follows Git-friendly structure with all assets self-contained and build tools optional.
